⚠️ WARNING redirect HTTP → HTTPS yönlendirmesi yok
HTTP versiyonundan HTTPS'e otomatik yönlendirme tespit edilemedi.		 Tüm HTTP trafiğini 301 redirect ile HTTPS'e yönlendirin.
⚠️ WARNING robots robots.txt erişilemiyor
robots.txt dosyasına erişilemiyor. Bu dosya arama motoru botlarına hangi sayfaları tarayabileceklerini söyler.		 Kök dizine uygun bir robots.txt dosyası ekleyin.
⚠️ WARNING sitemap XML Sitemap bulunamadı
sitemap.xml veya sitemap_index.xml erişilemiyor. Sitemap, Google'ın sayfalarınızı keşfetmesini hızlandırır.		 XML sitemap oluşturup /sitemap.xml adresinde yayınlayın.
⚠️ WARNING meta 1 sayfada meta description eksik
Meta description olmayan sayfalarda Google kendi snippet'ini oluşturur.		 Her sayfaya 120-160 karakter arası özgün meta description yazın.
⚠️ WARNING schema Schema markup kapsamı düşük (%0)
1 sayfanın sadece 0'inde schema var. Schema markup SERP'te rich snippet gösterimini sağlar.		 Ürün sayfalarına Product, kategori sayfalarına ItemList, blog yazılarına Article schema ekleyin.
⚠️ WARNING content 1 sayfada thin content var (<300 kelime)
Az içerikli sayfalar Google'ın 'Helpful Content' filtresine takılabilir. Ortalama kelime sayısı: 21		 Thin content sayfalarını en az 500 kelimeye çıkarın, veya ilgili sayfalarla birleştirin.
⚠️ WARNING images Görsel alt text kapsamı düşük (%0)
0 görselden 0'inde alt text yok. Alt text hem erişilebilirlik hem Google Görseller SEO'su için önemli.		 Tüm görsellere açıklayıcı alt text ekleyin. Format: '{Ürün adı} {özellik}' (örn: '205/55R16 Michelin lastik')
⚠️ WARNING links Ortalama iç link sayısı düşük (0.0)
Az iç link, crawl bütçesinin verimsiz kullanılması demektir.		 Her sayfada en az 5-10 ilgili iç link bulundurun.
